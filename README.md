# Study zustand state manager

### Стек:

- React, TS, CSS modules, Zustand, React router, Vite, ES lint, axios, ant-design

## Установка и запуск

Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run dev
```

## Сборка

```
npm run build
```

## Просмотреть prod версию

```
npm run build
```

```
npm run preview
```
